| 121 | [Best Time to Buy and Sell Stock](https://leetcode.com/problems/best-time-to-buy-and-sell-stock/) | Return the max possible profit from buying stocks in one day and selling them in another based on the given array of prices. Only single purchase and selling are allowed. | Precomputed array: 1. Get array with max values on the right side.\n2.Then, iterate from the left and check the difference for every number with prices[i] and precomputed[i+1]. Get the max.\nTwo pointers: 1. Use pointers to buy (i) and sell (i+1). If difference is negative, increment buy and sell to the next pair. 2. If the difference is positive, slide only sell and buy will find the lowest value. Get the max. O(n) |

Keep the original order. | Two pointers: to check (i) and write (k) numbers 1. If a[i] == a[i-1], just increment i.\n2. If a[i] != a[i-1], write a[i] to a[k] and i++, k++. O(n) |
